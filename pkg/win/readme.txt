DualWord-index - open-source cross-platform Xapian index viewer.

With its help you can:
	-browse documents.
	-view and sort a list of terms.
	-perform search using default Xapian query parser, no stemming.

How to install:
	-verify the checksum of the zip file.
	-unzip and run dualword-index.exe.
	-you can open "test-index" directory to test dualword-index.
